# Cryptoelectro-au 🛒

**[cryptoelectro.au](https://cryptoelectro.au)** — Australia's premium electronics marketplace with cryptocurrency payments.

## Features

- 🛍️ Premium electronics: smartphones, cameras, computers, gaming consoles
- 💰 Pay with Bitcoin, Ethereum, USDT, TRX, and 100+ cryptocurrencies
- 🚀 Fast shipping Australia-wide & surrounding regions
- 🤝 Affiliate program — earn 5% commission in crypto
- 🎁 Loyalty rewards system (Bronze → Diamond)
- 🔒 Secure payments via NowPayments

## Tech Stack

- **Frontend**: Next.js 16, TypeScript, Tailwind CSS, Framer Motion
- **Backend**: Next.js API Routes, Prisma ORM
- **Database**: PostgreSQL (Neon)
- **Payments**: NowPayments (BTC, ETH, USDT, TRX, SOL...)
- **Auth**: JWT with HTTP-only cookies
- **Deployment**: Vercel
